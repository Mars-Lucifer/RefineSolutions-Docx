Это докуменетация студии Refine Solutions для работников и других нуждающихся лиц. В ней описано множество информации по которым разрабатываются системы внутри нашей студии.

В этом Readme будет описана структура документации, ссылки на важные материалы, первостепенная информация про документацию например типы записей.

## Категории записей

Поскольку документация планирует включить очень большое количество разной информации со всей студии, категоризация записей особо важная деталь чтобы облегчить поиск и дальнейшее масштабирование. Записи разделены на:
1. Проблемно-ориентированные
Признаки: Конкретная проблема → причина → решение.
Цель: Быстро решить повторяющиеся технические проблемы.
Где: В `faq/` разделах.
